# Simulador-de-colisoes-cpp
Simulador de colisões da APS de Física I do professor Barreto, o programa gera bolinhas com velocidades e massas aleatórias


# Como executar o projeto:
Baixe o executável correspondente ao seu sistema operacional (Windows ou Linux) e execute preferencialmente pelo terminal ou cmd, você pode criar um arquivo de entrada e definir parâmetros para a geração das bolinhas (veja entrada.txt como exemplo), se você baixar a versão para o Windows, o exectável deverá estar junto com a SDL3.dll <br><br>
Parâmetros de entrada.txt: <br> <ul>
<li> quant_bolas: quantidades de bolas a serem geradas 
<li> dt: intervalo de tempo considerado para simulação 
<li> tam_x: largura da janela (e da mesa de sinuca) 
<li> tam_y: altura da janela (e da mesa de sinuca)
<li> v_min: velocidade mínima que as bolinhas podem ser geradas 
<li> v_max: velocidade máxima que as bolinhas podem ser geradas 
<li> m_min: massa mínima que as bolinhas podem ter 
<li> m_max: massa máxima que as bolinhas podem ter 
<li> r_bola: raio de cada bolinha </ul>
